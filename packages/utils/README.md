# @aozi6666/bee-utils

Front-end utilities for Bee Design.

## Install

```bash
pnpm add @aozi6666/bee-utils
```

## Exports

- `clamp(value, min, max)`
- `pick(obj, keys)`
- `omit(obj, keys)`
- `mergeRefs(...refs)` (React refs helper)
- `composeEventHandlers(theirHandler, ourHandler, options?)`
- `getScrollParent(element, boundary?)`
